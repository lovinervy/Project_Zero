Сервис предназначен для автоматического перевода и закадрового озвучивания видео из видеохостинга YouTube.
Для работы добавьте ссылку на видео из YouTube в поле вводе url, затем нажмите на кнопку отправить.
Затем выберите параметры перевода и снова нажмите отправить.
Страница будет долго обновляться т.к. будет происходить обработка видео, как только страница обновится появиться плеер с видео.
